# nodejs

## 安装NVM

windows下推荐使用[nvm-windows](https://github.com/coreybutler/nvm-windows)

Linux和macOS下推荐使用[nvm](https://github.com/nvm-sh/nvm)

## 使用NVM安装Node.js

```bash
nvm install <version>
nvm use <version>
```

## 全局安装包

```bash
npm install -g <package-name>
```

1. pnpm: 

```bash
npm install -g pnpm
```

2. yarn:

```bash
npm install -g yarn
```

3. typescript:

```bash
npm install -g typescript
```

4. ts-node:

```bash
npm install -g ts-node  
```
